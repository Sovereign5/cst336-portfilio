Swift: Apple's Definitive Language

Swift is the general-purpose language for all iOS, macOS, and watchOS devices provided by Apple. Created in 2014, Swift is a modern programming language most similar to other modern languages, like Python. Apple previously used Objective-C as their proprietary language for all devices, however Swift has replaced it.
